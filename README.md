# CryptoHack - 🔥 CryptoHack.org — 필수 암호학 실습 플랫폼 (Blockchain Security Cryptography Lab)
URL: https://cryptohack.org/courses/ / [https://blog.cryptohack.org/new-challenges-july-2024****](https://blog.cryptohack.org/)
<br>
CryptoHack은 블록체인 보안 연구자들이 사용하는 최고 수준의 실습형 암호학 플랫폼이다.
비트코인 Script 분석, ECDSA 공격, 해시 기반 보안, ZK·PQC 준비까지 커버하는 문제들이 포함된다.
연습하는 내용은 비트코인·이더리움·ZK-Proofs·PQC·해시 함수·서명 알고리즘 공격의 이해를 폭발적으로 높여준다.

비트코인 Script → 서명 → 해시 → 영지식 → PQC
이런 구조를 이해하기 위한 초·중·고급 난이도 코스가 완전히 정렬돼 있어.


🧩 CryptoHack의 장점
✔ 1) 블록체인 보안을 위한 “근본식” 암호학을 실전으로 배운다

비트코인 Script 해킹도, ECDSA 취약점 분석도, ZK-proof도
결국 수학 + 암호학 + 변조 공격 기반이다.

CryptoHack은 그 근본을 훈련시키는 곳.

✔ 2) 단계별 난이도 (입문 → 고급 → 전문가)

Modular arithmetic

RSA·ECC 해킹

해시 충돌·엑스플로잇

Padding oracle

ECDSA nonce leak

Lattice attacks (PQC 관련)

Zero-knowledge precursor 수학

✔ 3) ZK 관련 준비에도 최고

ZK Hack을 잘 하려면
modular 연산, group theory, hash permutation, commitment 구조를 알아야 하는데
CryptoHack이 그걸 실전 문제로 훈련해준다.

✔ 4) PQC(양자 안전 암호) 배우는 데 도움

현재 CryptoHack은 PQC 문제도 확장 중이라
SPHINCS+, Dilithium 같은 구조를 이해하는 데 기초 체력을 만들어 준다.

✔ 5) 실제 CTF에서 나오는 문제 스타일과 동일

Trail of Bits

Paradigm CTF

ZK Hack

Google CTF

DragonCTF

등에 동일한 유형이 등장한다.
